## Overview
`hzt.js` is the intersection of high-stakes engineering and raw artistic precision. It is an ultra-lightweight (1.4KB), zero-config JavaScript library designed to build premium, cinematic web experiences. No Virtual DOM overhead, just pure GPU-accelerated rendering.

* ⚡ **Zero Dependencies:** Pure Vanilla JS. No React, no jQuery, no bloat.
* 🛡️ **Secure Architecture:** Built-in sanitization preventing CSS injection & XSS.
* 🎨 **Theme API:** Inject custom color protocols effortlessly.
* 🧱 **Component Registry:** Modular layout injection for collaborative environments.
